# Flexcil Backup Viewer — Web Version

The web source lives in `artifacts/flex-viewer/` at the repository root.

Run the web app locally (from repo root):
```bash
pnpm install
pnpm --filter @workspace/flex-viewer run dev
```

For the native Android app, see the root-level `app/`, `build.gradle.kts`, and `settings.gradle.kts`.
